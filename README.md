# Sales Report Dashboard (Power BI)

##  Overview
This project is an interactive Sales Report Dashboard built using Power BI. 
It analyzes sales data to provide meaningful business insights through visual reports and a structured data model.

##  Data Model
The dashboard is built using a relational data model including:
- Sales Table (Fact Table)
- Customer Table
- Product Table
- Date Table

Relationships are created between these tables to enable accurate filtering and analysis.

##  Features
- Monthly Sales Trend Analysis
- Sales by Due Date
- Order Quantity by Country/Region (Map)
- Category & Sub-Category Sales Breakdown
- Year & Month Filters (Slicers)

##  Tools & Technologies
- Power BI Desktop
- Data Modeling
- Excel  Dataset

##  Key Insights
- Identified highest revenue months
- Compared regional sales performance
- Analyzed top-selling product categories
- Observed yearly growth patterns
